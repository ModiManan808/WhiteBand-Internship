# Ethical Hacking Task 01 - Information Gathering & Reconnaissance

## Objective

The objective of this task is to understand the reconnaissance phase of ethical hacking by collecting publicly available information about a target using passive information gathering techniques.

## Target Information

* **Website Name:** OWASP Juice Shop
* **URL:** https://juice-shop.github.io/
* **Assessment Type:** Passive Reconnaissance

## Activities Performed

* WHOIS Lookup
* DNS Enumeration
* Website Technology Identification
* HTTP Security Header Analysis
* robots.txt Analysis
* sitemap.xml Analysis

## Tools Used

* WHOIS Lookup Service
* DNSChecker
* Wappalyzer
* SecurityHeaders.com
* Web Browser

## Files Included

* Reconnaissance Report (PDF/DOCX)
* Screenshots
* Research Notes

## Key Findings

* The website is hosted on GitHub Pages.
* Multiple CDN providers including Cloudflare and Fastly were identified.
* HTTPS enforcement is enabled through HSTS.
* Several recommended security headers were missing.
* No robots.txt or sitemap.xml files were found.

## Learning Outcomes

This task provided practical experience in passive reconnaissance techniques and demonstrated how publicly available information can reveal valuable insights about a target system while maintaining ethical and legal boundaries.
